# ⚾ 컴투스 프로야구 데이터 분석 시리즈 개요   
    
## 1. 이 시리즈를 시작하는 이유  
  
야구 게임을 하다 보면 이런 질문이 자연스럽게 생긴다.  
  
- 이 선수, 체감은 좋은데 수치로도 좋은 걸까?  
- 같은 OVR인데 왜 성능 차이가 크게 느껴질까?  
- 타자와 투수의 ‘강함’은 무엇으로 결정되는 걸까?  
  
컴투스 프로야구는 단순한 카드 게임이 아니라  
**수치, 확률, 메타가 결합된 데이터 기반 시뮬레이션 게임**이다.  

이 시리즈는 게임 플레이 경험에 의존하던 판단을 넘어,  
**데이터로 타자와 투수를 분석하고 해석하는 것**을 목표로 한다.  
  
---  
  
## 2. 분석 대상과 범위   
  
본 분석은 컴투스 프로야구 내 **타자·투수 카드**를 대상으로 한다.  
  
### 분석 대상
- 타자 카드 (파워, 정확, 선구, 인내, 주루, 수비)
- 투수 카드 (구속, 변화, 구위, 제구, 지구력, 수비)

### 분석 범위  
- 기본 스탯 구조 분석  
- 스탯 간 상관관계  
- 포지션·유형별 성능 차이  
- 체감 성능과 수치의 괴리  
- 메타 변화에 따른 가치 변화  
  
※ 특정 시즌/카드에 국한되지 않고, 가능한 한 **구조적 관점의 분석**을 지향한다.  
  
---
  
## 3. 분석 관점과 접근 방식    
  
이 시리즈는 단순한 스탯 나열이 아니다.  
다음과 같은 질문을 중심으로 분석한다.   

### 타자 분석 관점
- 컨택 vs 파워의 실제 기여도는 어떻게 다른가?  
- 선구 수치는 출루에 얼마나 영향을 주는가?  
- 스피드는 체감 성능에 얼마나 반영되는가?  
- 포지션별로 중요한 스탯은 무엇인가?  

### 투수 분석 관점
- 구속·제구·변화구 중 가장 영향력이 큰 요소는?  
- 구종 개수와 성능의 관계  
- 선발과 불펜의 스탯 해석 차이  
- 체력 수치의 실제 의미  

모든 분석은 **“게임 로직상 이 수치가 어떤 역할을 하는가”**를 중심으로 진행한다.  

---

## 4. 데이터 분석 방법  
  
분석은 다음과 같은 흐름으로 진행된다.  
  
1. 카드 스탯 데이터 정리  
2. 파생 변수 생성 (비율, 합성 지표 등)  
3. 통계적 분석  
   - 분포 분석  
   - 상관관계  
   - 그룹 비교
4. 시각화를 통한 패턴 해석  
5. 실제 게임 체감과의 비교  
  
데이터 분석 도구로는 Python 기반 분석 환경을 활용하며, 시각화와 해석을 중심으로 설명한다.  
  
---
  
## 5. 앞으로 다룰 콘텐츠 예고    

이 시리즈에서는 매주 다음과 같은 주제를 다룰 예정이다.  

- 타자 핵심 스탯 구조 분석  
- 투수 구종 구성과 성능 관계   
- “체감이 좋은 선수”의 데이터적 특징  

각 글은 **하나의 질문 → 데이터 분석 → 해석 → 정리** 형태로 구성된다.  
  
---

이 시리즈는 고수만을 위한 분석도, 초보를 위한 공략도 아니다.  
  
- 왜 이 선수가 강한지 알고 싶은 사람
