# 🦥 Machine Learning 

## 개념  
1. 인공지능 : 인공적으로 지능을 만드는 것  
  - HOW : 점진적인 **학습 알고리즘을 통해 스스로를 개선**하여 데이터가 프로그래밍을 수행하도록 지원  
  - WHY : 왜 인공지능을 사용하려고 할까?  

  - 규칙기반 학습과 추론  
    - 여러번 데이터를 통해 **규칙을 도출**하여, **새로운 데이터에 대해서도 자동화/최적화** 하기 위해  

2. 머신러닝 : 기계가 인간의 개입없이 스스로 학습하는 기술  
  - HOW : 데이터와 정답을 **지속적으로 전달**하여 **스스로 원리를 학습**하도록 진행  
  - WHY : 그렇다면 왜 규칙기반 말고 머신러닝이 나오게 되었을까?  

  - 머신러닝의 학습과 추론  
    - 여러번 데이터를 **학습시켜**, **스스로 추론**하도록 만들어 **자동화/최적화** 하기 위해  
    - 피처를 직접 지정해줘야함  

3. 딥러닝 : 머신러닝 중에 신경망을 통해 깊게 쌓은 모델을 사용하는 기술  
  - HOW : 데이터와 정답을 **지속적으로 전달**하되, **인간의 뇌를 모방하여 깊이쌓은 신경망 모델** 사용
